# HTML Resume Page

This project is a single-page resume website built using **only HTML**.  

 Project Overview

- Single-page resume built using plain HTML
- No CSS or JavaScript used
- Focus on semantic HTML elements like:
  - header
  - section
  - headings
  - lists
  - tables

The resume includes:
- Header with name and role
- About Me section
- Skills list
- Experience table
- Projects section
- Education table
- Contact details


 Setup Steps (How to run locally)
    'BASH'
1. Clone the repository:

   git clone https://github.com/your-username/html-resume-page.git
   
2. cd html-resume-page

3.use code . to open in VS Code 
   
4. Initialize git repository
    git init
5. Add files using git add .
6. Commit using commit -m "commit message"
7. git push
